# Documentation

This directory hosts the Sphinx documentation for `pytopoviz`.

## Quickstart

```bash
pip install -e ".[docs]"
make html
```

Built docs will be available under `docs/_build/html/index.html`.

## Contents

- `index.rst` – root document.
- `conf.py` – Sphinx configuration mirroring the defaults used in `pytopotoolbox`.
- `_static/` – place custom assets here.
